iPhone版のインストール

App Store 不要。約30秒・3タップでホーム画面からアプリのように使えます。

Safari で開いてください
LINE や Instagram などのアプリ内ブラウザでは「ホーム画面に追加」が使えません。下のボタンで Safari に移動してください。

インストール済みです

ホーム画面の「千里の道」アイコンから開いてください。次回からはワンタップで起動できます。

アプリを開く

かんたんインストールおすすめ

TestFlight も App Store も不要。Safari からホーム画面に追加するだけです。

1 画面下（または上）の ⬆️ 共有 ボタンをタップ
2 メニューから 「ホーム画面に追加」 を選ぶ
3 右上の 「追加」 をタップ → 完了
Safari で開く

ネイティブ版（TestFlight）

歩数の自動記録がより安定します。配信開始後、下のボタン1つでインストールできます。

TestFlight でインストール

※ 現在準備中です。公開後、このページのボタンからインストールできます。

インストール後

  1. ホーム画面の 「千里の道」 アイコンをタップ
  2. 初回のみ モーション（歩数） の許可を「許可」
  3. 「はじめる」をタップすれば、すぐに旅を始められます

よくある質問

Android との違い
iPhone のホーム画面版は、画面を開いている間に歩数を記録します。ネイティブ版（TestFlight）公開後は、より自動的に記録されます。

アップデート
Web 版は自動で最新になります。ネイティブ版は TestFlight から更新してください。

歩数が記録されない
設定 → プライバシーとセキュリティ → モーションとフィットネス → 「千里の道」をオンにしてください。

Android の方
Android版インストールページ をご利用ください。
