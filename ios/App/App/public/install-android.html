Android版のインストール

Android版（ネイティブアプリ）をインストールすると、画面を消していても歩数が自動記録されます。Play ストアは不要です。

Android版をダウンロード

※ 初回のみ「提供元不明のアプリ」の許可が必要です（下記手順参照）

インストール手順

  1. 上のボタンから APK ファイルをダウンロード
  2. ダウンロード完了後、通知または「ファイル」アプリから APK を開く
  3. 「提供元不明のアプリ」許可を求められたら 許可 する（設定画面が開く場合あり）
  4. 「インストール」をタップ
  5. インストール後、アプリを開き 身体活動 と 通知 を許可
  6. 「はじめる」をタップして計測を開始
  7. 通知バーに「万歩計 計測中」と表示されれば、常時記録が有効です

よくある質問

ブラウザ版との違い
Chrome やホーム画面追加版は、画面を開いている間だけ記録します。常時記録には Android版（APK）が必要です。

アップデート
新機能を使うときは、同じページから最新版 APK をダウンロードして上書きインストールしてください。

歩数が記録されない場合
設定 → アプリ → 千里の道も一歩から → 身体活動・通知をオンにしてください。バッテリー最適化をオフにすると安定します。

iPhone
iPhone版インストールページ から、Safari で「ホーム画面に追加」できます（App Store 不要・約30秒）。
